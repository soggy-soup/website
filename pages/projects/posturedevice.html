Posture device

7.01.25 - 7.07.25

I desigend and built a wearable that uses a 6 DOF IMU to track a users posture and provides haptic feedback with an eccentric motor when the user is slouching. The device is enclosed in a 3D printed TPU case that comprises the mcu is an ESP32-C3, everything is powered by a 3.3V battery.
